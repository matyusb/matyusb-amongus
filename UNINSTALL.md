# Restore or remove the modded installation

Close Among Us, Steam, Epic Games Launcher, and EpicGamesStarter before changing files.

## Desktop copy

The installers create a separate folder named `Among Us - TOU Mira` on the desktop. Delete that folder to remove the desktop copy. A previous copy may have been renamed with `regi-YYYYMMDD-HHMMSS`; inspect it before deleting it.

## Steam installation

The Steam installer also modifies the configured Steam Among Us folder.

1. In Steam, open **Among Us → Properties → Betas** and return to the normal/default branch when appropriate.
2. Use Steam's **Verify integrity of game files** option, or uninstall and reinstall Among Us.
3. Remove any leftover mod files only after confirming that Steam has restored the official game.

## Epic Games installation

The Epic downgrader normally creates an installation under `%USERPROFILE%\Games\AmongUs`, and the installer may copy it to the desktop.

Delete only the separate downgraded/modded folder created by the process. Reinstall or verify the official Epic Games installation through the Epic Games Launcher when needed.

Do not delete unrelated folders under `%USERPROFILE%\Games`.
